Recipe Swap

All the links containing the API EndPoints. just copy and paste.

Endpoints

      get all users : /users/register
      post user Register : /users/register
      post user Login : /users/login
      post user forgot : /users/forgot
      post user reset (email link) : /users/resetpassword/{token required}
      get user Logout : /users/logout
      delete user and his/her recipes: /user/delete
      get All recipes : /recipe

    Protected Routes

      get user recipes : /recipe/profile
      post new recipe : /recipe/add
      Update recipe : /recipe/update/{recipeId required}
      Delete recipe : /recipe/delete/{recipeId required}
